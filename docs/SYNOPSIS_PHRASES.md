# Synopsis Phrases

Reusable phrases for the synopsis template.

- Open Questions / To clarify
  "The following items require clarification before protocol finalization:"

- HOSPITALIZATION_DURATION
  "Confirm hospitalization duration, sampling duration, and follow-up duration (t1/2 coverage)."

- PHONE_FOLLOW_UP
  "Confirm whether phone follow-up is acceptable or recommended."

- BLOOD_VOLUME
  "Confirm blood volume constraints (total vs PK-only)."

- DEC85_REQUIRED_PK
  "Provide missing PK parameters required by Decision 85 (Cmax, AUC0-t, AUC0-inf, t1/2 or lambda_z)."

- DQI_RED
  "Confirm sources / provide stronger BE/PK evidence."

- CV_DERIVED_ASSUMPTIONS
  "Confirm assumptions for CV derived from CI (90% CI, 2x2 crossover, log-scale, correctness of n/CI)."

- CV_RANGE_UNCERTAIN
  "Provide measured CVintra if possible; risk-based N used."

- CV_CONFIRM_NDET
  "Confirm CV to enable N_det calculation."
